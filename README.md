# VoiceBridge – Assistive Communication App

 Secured First Prize in Project Expo

## Overview
VoiceBridge is an Android application designed to support communication for deaf and visually impaired users. It combines voice, text, and sign language features into a single platform to improve accessibility and real-time interaction.

## Workflow
###  User Authentication
- Users can register and log in using email and password
- After login, users are directed to the main dashboard
###  Dashboard Modules
The dashboard includes six main features:
#### Video Call
- Real-time video communication using Jitsi SDK
####  Learning Mode
- Learn sign language using 300+ GIF-based avatars
- Includes search functionality for easy access
####  Chat Translation
- Translate text into 8–10 different languages
- Helps users communicate across language barriers
####  SOS Emergency
- Sends an emergency message with live location
- User selects a contact and triggers alert instantly
####  Speech to Avatar
- Converts spoken words into sign language GIFs
- Enables real-time communication between deaf and non-deaf users
#### Games
- Memory Game: Improves memory by repeating sequences
- Sign Matching Game: Match words with correct sign language GIFs

##  Tech Stack
- Java (Android Development)
- Android Studio
- Firebase (Authentication & Realtime Database)
- Jitsi SDK
